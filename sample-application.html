3. Sample Application

The sample is a number guessing game in Python (source code above in basic elements).

Design of System Features with Flowchart and Description

Flowchart (text-based):

Start
  |
Generate random number (1-100)
  |
Loop: Input user guess
  |  
Compare guess to number
 /  \  
Too low? -> Print "Too low!" -> Increment attempts -> Loop
Too high? -> Print "Too high!" -> Increment attempts -> Loop
Correct? -> Print "Correct! Attempts: X" -> End

Description: The system generates a random number, takes user inputs in a loop, compares them, provides feedback, and ends when correct.

Output of Each System Feature

  • Random number generation: Internal, no direct output.
  • User input: Prompts "Guess a number between 1 and 100: " and reads input.
  • Comparison: Outputs "Too low!", "Too high!", or "Correct! It took X attempts."
