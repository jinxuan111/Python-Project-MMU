4. Short Training in Less Than 15 Minutes

This quick demo uses the sample number guessing game to show basic elements. Time estimate: 10 minutes.

  1. (2 min) Install Python if needed (from python.org). Open a terminal or IDE.
  2. (3 min) Copy the code into a file 'game.py'. Run it: python game.py. Play the game to see outputs.
  3. (3 min) Explain names/bindings: Change 'attempts = 0' to see dynamic binding.
  4. (2 min) Expressions: Modify 'attempts += 1' to 'attempts = attempts + 2' and rerun.
  5. (2 min) Control: Remove 'break' to see infinite loop on correct guess.
  6. (2 min) Subprograms: Call the function multiple times by wrapping in a loop.

This demonstrates Python's basics interactively.
