2. Basic Elements of Python (with Source Codes from Sample Application)

The sample application is a simple number guessing game. Full code:

import random

def guess_number():
    number = random.randint(1, 100)
    attempts = 0
    while True:
        guess = int(input("Guess a number between 1 and 100: "))
        attempts += 1
        if guess < number:
            print("Too low!")
        elif guess > number:
            print("Too high!")
        else:
            print(f"Correct! It took {attempts} attempts.")
            break

guess_number()

Names and Bindings (Lecture 6)

Names like 'number', 'attempts', and 'guess' are bound to values dynamically. Python uses late binding, where types are checked at runtime. Example: number = random.randint(1, 100) binds 'number' to an integer.

Expressions and Assignment (Lecture 7)

Expressions include arithmetic like attempts += 1 (assignment with addition). Assignment: guess = int(input(...)) converts input to int and assigns.

Control Structures (Lecture 8)

Uses a while loop: while True: for indefinite iteration, and conditional: if guess < number: ... elif ... else: ....

Subprograms (Lecture 9)

The function def guess_number(): is a subprogram that encapsulates the game logic. It's called at the end: guess_number().
